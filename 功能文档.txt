================================================================================
                        赛博除怨局 (Cyber Grudge) — 功能文档
================================================================================

【产品概述】

  赛博除怨局是一款赛博朋克风格的在线解压互动应用。
  用户输入被他人举报或针对的"黑状"，通过点击 SVG 小人特定身体部位"扎针"，
  触发物理震动动画，并获得一句由 AI 生成的讽刺咒语作为精神补偿。

【核心功能】

  1. 告密案卷输入
     - 在左侧文本区输入黑状内容（限 100 字）
     - 点击"锁定罪证"按钮将事件固定，锁定后 textarea 变为只读

  2. 锁定罪证仪式
     - 锁定前小人处于暗淡不可点击状态
     - 锁定后小人绿色霓虹发光变亮，进入可交互状态
     - 营造"证据确凿、开始行刑"的仪式感

  3. SVG 小人扎针交互
     - 头部：点击后眼睛变成蚊香圈 + 360° 旋转
     - 嘴巴：弹性放大扭曲（管住嘴！）
     - 左手/右手：关节抽搐颤动（管住手！）
     - 腿部：扭曲变形（别跑！）
     - 每次有效点击触发全屏震动

  4. AI 赛博咒语生成
     - 默认模式：随机返回本地预置的讽刺咒语
     - 深度除怨模式（需填写 DeepSeek API Key）：调用大模型实时生成更精准的咒语
     - 咒语以气泡形式弹出显示，5 秒后自动消失

  5. BYOK（自带 Key）模式
     - API Key 仅保存在浏览器 localStorage，绝不上传至任何第三方服务器
     - 输入框为 password 类型，输入内容不可见
     - 切换开关可平滑展开/收起 Key 输入区域

  6. 实时弹幕与计数器
     - 顶部实时显示"今日灵山超度总数"
     - 水平滚动弹幕轨模拟其他网友的战报

【使用步骤】

  第一步：在左侧文本区输入黑状，例如"小明跟老板打小报告说我上班摸鱼"
  第二步：点击"锁定罪证"按钮，textarea 变红锁定
  第三步：（可选）打开"深度除怨"开关，输入 DeepSeek API Key
  第四步：点击右侧 SVG 小人任意身体部位
  第五步：观看 GSAP 动画 + 阅读诅咒气泡

【技术特点】

  - Next.js 14 App Router 全栈架构
  - GSAP 驱动的高性能 SVG 物理动画
  - Tailwind CSS 暗色赛博朋克主题
  - 支持 DeepSeek 大模型集成（BYOK 本地存储）
  - TypeScript 类型安全
  - 响应式布局（桌面端优先）

【本地运行】

  npm install
  npm run dev
  浏览器打开 http://localhost:3000
